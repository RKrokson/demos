# containers Reference

This directory contains 7 icons extracted from `containers.excalidrawlib`.

|-----------|----------|

| Azure Red Hat OpenShift | `icons/Azure-Red-Hat-OpenShift.json` |

| Container Instances | `icons/Container-Instances.json` |

| Service Fabric Clusters | `icons/Service-Fabric-Clusters.json` |

## Usage

Each icon JSON file contains the complete `elements` array needed to render that icon in Excalidraw.
You can copy the elements from these files into your Excalidraw diagrams.
