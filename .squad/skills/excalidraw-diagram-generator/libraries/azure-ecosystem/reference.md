# azure-ecosystem Reference

This directory contains 3 icons extracted from `azure-ecosystem.excalidrawlib`.

## Available Icons

| Icon Name | Filename |
|-----------|----------|
| Applens | `icons/Applens.json` |
| Azure Hybrid Center | `icons/Azure-Hybrid-Center.json` |
| Collaborative Service | `icons/Collaborative-Service.json` |

## Usage

Each icon JSON file contains the complete `elements` array needed to render that icon in Excalidraw.
You can copy the elements from these files into your Excalidraw diagrams.
