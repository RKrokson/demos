# blockchain Reference

This directory contains 6 icons extracted from `blockchain.excalidrawlib`.

## Available Icons

| Icon Name | Filename |
|-----------|----------|
| ABS Member | `icons/ABS-Member.json` |
| Azure Blockchain Service | `icons/Azure-Blockchain-Service.json` |
| Azure Token Service | `icons/Azure-Token-Service.json` |
| Blockchain Applications | `icons/Blockchain-Applications.json` |
| Consortium | `icons/Consortium.json` |
| Outbound Connection | `icons/Outbound-Connection.json` |

## Usage

Each icon JSON file contains the complete `elements` array needed to render that icon in Excalidraw.
You can copy the elements from these files into your Excalidraw diagrams.
