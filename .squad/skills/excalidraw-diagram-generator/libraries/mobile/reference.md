# mobile Reference

This directory contains 3 icons extracted from `mobile.excalidrawlib`.

## Available Icons

| Icon Name | Filename |
|-----------|----------|
| App Services | `icons/App-Services.json` |
| Notification Hubs | `icons/Notification-Hubs.json` |
| Power Platform | `icons/Power-Platform.json` |

## Usage

Each icon JSON file contains the complete `elements` array needed to render that icon in Excalidraw.
You can copy the elements from these files into your Excalidraw diagrams.
