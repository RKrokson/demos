# azure-stack Reference

This directory contains 7 icons extracted from `azure-stack.excalidrawlib`.

## Available Icons

| Icon Name | Filename |
|-----------|----------|
| Capacity | `icons/Capacity.json` |
| Infrastructure Backup | `icons/Infrastructure-Backup.json` |
| Multi Tenancy | `icons/Multi-Tenancy.json` |
| Offers | `icons/Offers.json` |
| Plans | `icons/Plans.json` |
| Updates | `icons/Updates.json` |
| User Subscriptions | `icons/User-Subscriptions.json` |

## Usage

Each icon JSON file contains the complete `elements` array needed to render that icon in Excalidraw.
You can copy the elements from these files into your Excalidraw diagrams.
